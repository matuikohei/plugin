# Minecraft プラグイン

**開発した背景**
---
私は現在、未経験からのエンジニア転職に向けてデイトラでJavaを学習しています。
このプラグインは、Javaの条件分岐やループ処理、入出力処理を学習するために作成しました。

**プラグイン内容**
---
* スニーク回数が偶数時に赤、青、白、黒の順番で４発花火が打ち上がる
* setPower(計算式)を使用し、花火の飛距離を設定
* 花火が打ち上がった際に「たーまやー」という文字が出力される
* ベットに入ると、アイテムが最大数まで増加する
